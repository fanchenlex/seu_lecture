在网页中右键或按 F12 进入控制台，复制请求头中的 cookie 到 main.py 中，参考[这篇文章](https://blog.csdn.net/boheliang99/article/details/122348239)。

这篇文章可能说得不清楚，我再提示一下：
！[image](https://i-blog.csdnimg.cn/direct/3410a19ff0fa4614bb4276b2de99d9f4.png)

2.寻找需要预约的讲座名称
